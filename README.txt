iOS PUSH-COMPATIBLE APP — Weekly $50 Investor

WHAT THIS ZIP CONTAINS:
- Fully iOS‑optimized PWA (works on iPhone Home Screen)
- Real Web Push support
- Complete Node.js push server
- Service worker + manifest configured for iOS

IMPORTANT — iOS REQUIREMENTS:
You MUST host this PWA on HTTPS.
You MUST run the included Node.js push server.
You MUST generate VAPID keys using:

  npx web-push generate-vapid-keys

Paste them into:
  - config.js (public key)
  - server.js (public + private key)

INSTALLING ON IPHONE:
1. Host the PWA folder on ANY HTTPS web host
2. Visit the site on Safari
3. Tap "Share"
4. Tap "Add to Home Screen"
5. You now have a fully installable iOS app
6. Tap "Enable Push Notifications"

Your server will push every Monday at 6:30 AM.
